Требуется разработать настольное приложение Windows, реализующее:

## 2. Назначение и цели создания системы

### 2.1 Вид автоматизируемой деятельности
Автоматизация проверки файлов на наличие вредоносного ПО с использованием агрегированного анализа множества антивирусных движков.

### 2.2 Перечень автоматизируемых процессов

#### 2.2.1 Для пользователя:
- выбор файла;
- отправка файла на VirusTotal;
- автоматическое отслеживание статуса анализа;
- получение итогового отчёта;
- визуальная индикация уровня угрозы.

### 2.3 Показатели эффективности

#### 2.3.1 Удобство использования
- запуск проверки файла: ≤ 10 секунд;
- интуитивный интерфейс.

#### 2.3.2 Оперативность получения информации
- результаты отображаются сразу после их готовности;
- максимум ожидания — 180 секунд.

#### 2.3.3 Надёжность и точность
- корректная обработка ошибок сети, API, пользователя;
- точность данных гарантируется использованием оригинального ответа API VirusTotal.

#### 2.3.4 Доступность
Приложение автономно, требуется только доступ в интернет.

## 3. Характеристики объекта автоматизации

### 3.1 Сведения о пользователях
Студенты, преподаватели и ИТ-специалисты СПб ГУАП, проверяющие подозрительные файлы.

### 3.2 Особенности процесса проверки
- необходимость анализа множеством антивирусов;
- сложность ручной загрузки на сайт VirusTotal;
- потребность в удобном представлении результатов.

### 3.3 Особенности объекта
- интеграция с асинхронным внешним API;
- простота и отсутствие необходимости дополнительного серверного ПО;
- акцент на безопасности и прозрачности результатов.

### 3.4 Цели внедрения
- повышение уровня кибербезопасности;
- сокращение времени проверки файлов;
- стандартизация подхода к анализу.

## 4. Требования к ИС

### 4.1 Требования к системе в целом
- обеспечение полного процесса: выбор файла → анализ → вывод отчёта;
- простой интерфейс;
- корректная обработка сетевых и API-ошибок;
- зависимость только от интернет-доступа.

### 4.2 Требования к ПО

#### 4.2.1 Клиентская часть
- WPF,
- C#,
- .NET Framework 4.7.2+,
- Windows 10/11.

#### 4.2.2 Функциональность
- загрузка файла;
- отправка на https://www.virustotal.com/api/v3/files;
- опрос статуса по analysis ID;
- парсинг JSON;
- отображение вредоносных / подозрительных / безопасных сканеров;
- визуальное выделение итогового вердикта.

### 4.3 Требования к техническому обеспечению
- ОС Windows 10/11;
- CPU: Pentium 4+ (рекомендуется i3+);
- RAM: 2–4 ГБ;
- дисковое пространство: 50 МБ;
- обязательный доступ в интернет.
